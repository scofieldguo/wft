1.接口说明

查询卡的使用记录。

方法定义：queryRecord.do

2.请求参数
属性 类型 最大长度 是否必填 说明
partner String - Y 合作方标识，由平台分配
op int - Y 卡类型：2电信
cno String - Y 卡号
pwd String - Y 卡密
startdate String - Y 开始日期，yyyy-MM-dd
enddate String - Y 结束日期，yyyy-MM-dd
timestamp String - Y 时间戳，yyyy-MM-dd HH:mm:ss
standby String 100 N 备用字段，请求响应中原样返回
3.响应参数
属性 类型 说明
result int 响应码，见附件
timestamp String 时间戳，yyyy-MM-dd HH:mm:ss
cno String 卡号
standby String 备用字段，请求响应中原样返回
recordlist JSONArray 卡使用记录列表，json串

recordlist数据结构如下：

属性 类型 说明
starttime String 开始时间，yyyy-MM-dd HH:mm:ss
endtime String 结束时间，yyyy-MM-dd HH:mm:ss
timelen int 使用时长，秒
